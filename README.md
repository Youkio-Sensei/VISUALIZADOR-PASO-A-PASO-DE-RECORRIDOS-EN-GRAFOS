# VISUALIZADOR-PASO-A-PASO-DE-RECORRIDOS-EN-GRAFOS
Este proyecto es una aplicación de escritorio desarrollada en Java que permite visualizar y analizar recorridos en estructuras de datos de tipo grafo. La aplicación facilita el aprendizaje de la Teoría de Grafos y algoritmos de búsqueda como Dijkstra, mediante una interfaz gráfica interactiva que muestra el recorrido paso a paso.
