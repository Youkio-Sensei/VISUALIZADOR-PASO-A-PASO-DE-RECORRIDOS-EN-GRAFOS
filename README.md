# VISUALIZADOR-PASO-A-PASO-DE-RECORRIDOS-EN-GRAFOS
Este proyecto es una aplicación de escritorio desarrollada en NetBeans en lenguaje Java que permite visualizar y analizar recorridos en estructuras de datos de tipo grafo. La aplicación facilita el aprendizaje de la Teoría de Grafos y algoritmos de búsqueda como Dijkstra, mediante una interfaz gráfica interactiva que muestra el recorrido paso a paso.

# Características
* Agregar vértices al grafo
* Eliminar vértices
* Agregar aristas con peso
* Eliminar aristas
* Visualización gráfica del grafo
* Registro de eventos (log)
* Cálculo del camino más corto

# Requisitos
Antes de ejecutar el proyecto necesitas:
Java JDK 8 o superior
NetBeans IDE (recomendado)
Sistema operativo:
Windows
Linux
MacOS

# Abrir el proyecto en NetBeans
Abrir NetBeans
Ir a: File → Open Project
Seleccionar la carpeta:
VisualizadordeGrafos
NetBeans cargará automáticamente el proyecto.
3. Ejecutar el proyecto
Dentro de NetBeans:
Click derecho al proyecto → Run

# Ejecutar el programa directamente
El proyecto incluye un ejecutable:
VisualizadordeGrafos(programa ejecutable).exe
Solo debes hacer doble clic para abrir la aplicación.
(Disponible para Windows)
O presiona: F6
